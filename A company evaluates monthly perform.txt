A company evaluates monthly performance of employees based on multiple criteria.
Each employee has the following data:

Field	Data Type	Description
employeeId	int	Unique identifier for employee
taskCompletion	int	Number of tasks completed in the month
attendance	int	Number of days present in the month (out of 30)
peerScore	int	Peer evaluation score (0–100)
managerScore	int	Manager evaluation score (0–100)

The company wants a class that:
Determines performance level based on tasks completed.
Assigns bonus points based on attendance.
Classifies peer evaluation.
Classifies manager evaluation.
Determines promotion eligibility using all metrics.

CLASS NAME
EmployeeEvaluator

FUNCTIONS & CRITERIA
1. determineLevel
public String determineLevel(int taskCompletion)
Criteria:
High → TaskCompletion ≥ 80
Medium → 50 ≤ TaskCompletion < 80
Low → TaskCompletion < 50
Must use if–else-if statements
Return the level as String

2. attendanceBonus
public int attendanceBonus(int attendance)
Criteria:
Attendance ≥ 28 → 20 points
25 ≤ Attendance < 28 → 10 points
Attendance < 25 → 0 points
Use if–else-if statements
Return int bonus points

3. evaluatePeer
public String evaluatePeer(int peerScore)
Criteria:
PeerScore ≥ 85 → Excellent
60 ≤ PeerScore < 85 → Good
PeerScore < 60 → Poor
Must use relational operators and conditional statements
Return evaluation as String

4. evaluateManager
public String evaluateManager(int managerScore)
Criteria:
ManagerScore ≥ 90 → Outstanding
70 ≤ ManagerScore < 90 → Satisfactory
ManagerScore < 70 → Needs Improvement
Must use conditional statements
Return evaluation as String

5. promotionEligibility
public boolean promotionEligibility(int taskCompletion, int attendance, int peerScore, int managerScore)
Criteria:
Eligible for promotion only if:
TaskCompletion ≥ 80
Attendance ≥ 26
PeerScore ≥ 70
ManagerScore ≥ 75
Use logical operators (&&) and conditional statements
Return true/false
